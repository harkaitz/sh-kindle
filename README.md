# KINDLE UPLOADER

Command line utility to send files to the kindle.

## Dependencies:

- pdfcrop (optional).
- mpack, msmtp.

## Help

kindle

    Usage: kindle [-Vwm] PDFs...
    
    Send books to my kindle.
    
      -V : Show configuration.
      -w : Open amazon kindle library.
      -m : Open amazon my kindle site.
      -c : Crop pdf before sending.
